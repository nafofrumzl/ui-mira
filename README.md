# ui-mira

A lightweight library for data visualization and analytics.

### Install
```bash
npm install ui-mira
```

### Usage
```javascript
import { UiMira } from 'ui-mira';

const data = [{ x: 1, y: 10 }, { x: 2, y: 20 }];
const chart = new UiMira('#container');

chart.render(data);
```

### Tests
